# 🏹 Hunter: Darkbound 🏹

Hunter: Darkbound es un emocionante juego 2D desarrollado en JavaScript con el motor Phaser. En este desafiante título, te pondrás en la piel de un intrépido cazador de monstruos que deberá sobrevivir a intensas oleadas de enemigos para alcanzar su objetivo final: derrotar al temible jefe de zona.

## 💀 Sobrevive y mejora tus habilidades 💀

Cada zona del juego tiene una duración de diez minutos, durante los cuales deberás enfrentarte a hordas de enemigos cada vez más fuertes. A medida que derrotes adversarios, ganarás experiencia y subirás de nivel, desbloqueando poderosas habilidades y mejoras que te ayudarán en la batalla.


## ⚔️ Enfréntate a épicos jefes ⚔️

Si logras resistir hasta el final del tiempo límite, aparecerá un feroz jefe de área. Derrotarlo será clave para desbloquear nuevas zonas y desafíos aún más difíciles. ¿Tienes lo necesario para sobrevivir y convertirte en el cazador definitivo?
<br>
## ⚙️ Instalación ⚙️

Para instalar y ejecutar el proyecto en tu máquina local, sigue los siguientes pasos:

### 1️⃣ Clonar el repositorio

Ejecuta el siguiente comando en tu terminal o utiliza un software como Gitkraken o GitHub Desktop para clonar el repositorio:

```bash
git clone <URL_DEL_REPOSITORIO>
```

### 2️⃣ Instalar dependencias

Dentro del directorio del proyecto, instala las dependencias necesarias con:

```bash
npm install
```

### 3️⃣ Iniciar el servidor local
Para levantar el servidor local, usa el siguiente comando:

```bash
npm run dev
```

### 4️⃣ Acceder al juego
Una vez que el servidor esté en ejecución, accede a la URL proporcionada en la consola para jugar.
